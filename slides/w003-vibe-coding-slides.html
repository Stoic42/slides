Prompt 入门 · Formula
06 / 09
Prompt 秘籍

Context + Task + Constraints

01
Context
背景信息。把图片拖进去，把日记粘贴，把想法写清楚
02
Task
任务就是你想做什么，说明白目标和结果
03
Constraints
约束条件，格式、风格、边距、效率……说清楚
"我要一个细长腿、动漫风的插图，背景有浅浅海浪…" = Magic ✨
Context + Task + Constraints = 魔法

实操现场 · Validation
08 / 09
【便利贴法则】

痛点验票

Step 1: Write
  • 左边：你的 Idea（产品名）
  • 右边：崩溃瞬间（即你的 Context）
Step 2: Check
  • 写不出右边的"崩溃瞬间"？
  • 警报：你的 Idea 太大了！
  • 立刻缩小范围，直到你能写出那个瞬间
你的痛点 = 最好的 Prompt 输入
AI 不仅是你的程序员，它首先是你的镜子
