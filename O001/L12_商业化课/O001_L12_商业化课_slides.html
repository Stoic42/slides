02 / 08 · VALUE

为什么商业化比产品更难

黑客松做完 Demo，有团队，有媒体报道——但就是没有付费用户。

赵迪说："MVP 不是缩小版的产品，是缩小版的价值验证。"

很多人以为 MVP = 功能少一点。其实 MVP = 先验证用户真的愿意付钱。

MVP 陷阱

  • 功能做了一大堆 → 没人用
  • 用户说"不错" → 但不付费
  • Demo 很精美 → 商业化路径没想清楚

先有付费意愿，才有产品

讲师提示
问同学："你们觉得 MVP 是什么？" → 让2-3人回答
引出赵迪的核心观点：MVP 是价值验证，不是功能集合
停顿强调："付费意愿比功能完整更重要"

03 / 08 · VALIDATION

赵迪：需求验证四步法

01

找到愿意聊的人

不是随便问，是找真的有过这个痛点的人。朋友不算。

02

问痛点，不问方案

"这个问题有多痛？" 比 "你觉得这个功能好不好？" 更重要。

03

问钱，不问意愿

"你愿意为这个付多少钱？" 比 "你会用吗？" 更真实。

04

验证支付渠道

中国用户习惯微信/支付宝；海外用户习惯 Stripe/PayPal。支付方式决定转化率。

讲师提示
四步可以打乱顺序讲，重点是第3步"问钱"
强调：问"愿不愿意付费"没用，要问"愿意付多少"
案例：赵迪 可梦 的真实用户访谈案例（如果有）

04 / 08 · FOUNDER

Vivian：女性做产品的独特优势

"女性 founder 的 Demo 完成度往往更高，但获得的关注远少于男性。"

Vivian 的核心观点

  • 痛点优先：先确认这个痛点真实存在，再动手
  • 女性更懂女性用户：自己就是目标用户，更容易发现真实需求
  • 不要追 VC 喜欢的词：researcher founder / FDE / harness —— 先把产品讲清楚
  • 先有 traction，再讲故事：数字比叙事更重要

Vivian 特别提醒：奇绩喜欢的词你都没套上——researcher founder、FDE、harness。先把"为谁做什么事"讲清楚。

讲师提示
停顿 → "Vivian 是奇绩被投 CEO，她观察了大量早期项目"
重点讲"女性更懂女性用户"——这是 SheNicest 的核心叙事
强调：不要追风口词，先把 traction 做出来

05 / 08 · PITCH

Angel：Pitch 给投资人 + 自信心

Pitch 的核心结构

部分内容时长
1你是谁30秒
2你解决了什么问题1分钟
3你的解决方案1分钟
4 traction + 数据1分钟
5你需要什么30秒

Angel 的 BP 结构建议

  • 一句话定位：第一页就说清楚你是谁、做什么
  • 痛点要具体：不是"大家都有这个问题"，是"某类人在某场景下的具体痛苦"
  • 数据说话：没有 traction 就先展示用户访谈结果
  • 团队页要真诚：不是所有人都要 CTO，名片要真实

自信心比完美更重要

讲师提示
带着同学快速过一遍 BP 结构 → 可以打开一个真实 BP 当案例
强调"自信心比完美"——很多女性 founder 过度准备、害怕不完美
停顿：问同学"你们觉得自己现在最缺的是自信还是内容？"

06 / 08 · PAYMENT

赵迪：支付渠道与商业化路径

CN

国内渠道

  • 微信小商店（最常用）
  • 有赞 / 微店
  • 抖音小黄车
  • 支付宝收款码（最简单）
GLOBAL

海外渠道

  • Stripe（最推荐）
  • PayPal Business
  • Gumroad（创作者经济）
  • Product Hunt
COMMUNITY

平台合作

  • SheNicest 平台（奖励型众筹）
  • 小红书店铺
  • 摩点 / 众筹平台
  • 私域微信转账

赵迪提醒：支付方式决定转化率。中国用户习惯微信/支付宝，海外用户习惯 Stripe。先想清楚用户在哪个平台，再决定产品形态。

讲师提示
快速过一遍每个渠道的特点 → 重点推荐 SheNicest 平台
强调：私域转账无法规模化，平台是下一步
可以问："你们有同学已经开始变现了吗？" → 引出真实案例

07 / 08 · CHECKPOINT

商业化自检清单

做完 Demo 之后，问自己这 5 个问题

#问题验证方式
1谁是我的第一笔钱？访谈10个真实用户，问愿意付多少
2他们在哪个平台付款？微信小商店 / Stripe / 私域
3定价多少？不低于成本，先跑起来再说
4第一批用户从哪来？社群 / 小红书 / 黑客松社群
5有没有 MVP 以外的需求？访谈中问："你还想要什么？"
讲师提示
带同学一个个过 → 可以让同学自己填
第1题最难：很多人答不上来 → "那你还没准备好商业化"
第5题最容易被忽略：用户会主动告诉你下一步做什么
